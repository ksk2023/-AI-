# SpaceX（Space Exploration Technologies Corp.）数据可得性说明（2026-07-01）

## 二、数据源实测结果

本研究采用 Yahoo Finance 官方 API（cookie+crumb 认证）作为价格数据权威源。
对 SpaceX 所有可能的公开交易代码进行了逐一探测，结果如下：

| 候选代码 | 实测结果 | 实际归属 |
|---|---|---|
| `SPACEX` | HTTP 404（无此证券） | — |
| `SPACE` | HTTP 404 | — |
| `STARLINK` | HTTP 404 | — |
| `SPX` | 无有效行情 | 标普500指数的混淆代码 |
| `SPCE` | 有数据 | **维珍银河 Virgin Galactic**，非 SpaceX |
| `RKLB` | 有数据 | **Rocket Lab**，非 SpaceX |
| `SPCX` | 有数据 | SPCX Corporation（SPAC），非 SpaceX |

**结论：Yahoo Finance 不存在 SpaceX 的公开交易行情数据。**

## 三、SpaceX 私有状态事实依据

SpaceX 自 2002 年成立以来始终为私有公司，其主要事实依据包括：

1. **从未向 SEC 提交 S-1 注册声明**（IPO 必备文件），SEC EDGAR 系统中无 SpaceX 的公开募股记录。
2. **马斯克多次公开表态**（X/Twitter 及公开访谈）短期无 IPO 计划，且曾表示"Starlink 业务分拆上市前需先证明现金流可承受"。
3. **股权流动性来自二级市场私募交易**（如 LangX、Rainey 等私募股权交易平台上的员工老股转让），这些交易：
   - 无公开交易所撮合价格
   - 无逐日盯市（mark-to-market）的连续 OHLC 行情
   - 估值依据定期融资轮次（如 2024 年最新一轮估值约 1800-2500 亿美元区间，具体以公开报道为准）

## 四、对本研究的影响

由于缺乏连续、公开、可验证的日线价格序列，**无法对 SpaceX 进行标准的历史回测分析**（CAGR、MDD、波动率、夏普等指标均无法计算）。强行用任何其他代码的数据冒充 SpaceX 都将违反本研究"数据必须准确无误"的核心原则。
