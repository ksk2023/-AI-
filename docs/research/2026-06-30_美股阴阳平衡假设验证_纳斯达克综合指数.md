# 美股阴阳平衡假设验证报告（2026-06-30）

## 一、研究问题

**假设**：纳斯达克综合指数在每一个自然年之中,上涨的天数与下跌的天数应该大致相等（接近 1:1）,即"阴阳平衡"。

本报告用真实历史数据逐年检验这个假设。

## 二、数据来源

- **指数**：纳斯达克综合指数(NASDAQ Composite, Yahoo 代码 `^IXIC`),全球科技与成长股最广泛使用的基准指数。
- **数据源**：Yahoo Finance 官方 API,通过 cookie+crumb 认证绕过限流。
- **时间跨度**：1971-02-05（指数创立日,起始 100 点）至 2026-06-29,共 56 年（部分年份数据从该年首个交易日开始）。
- **原始数据文件**：`data/raw/daily_NASDAQ_COMPOSITE.parquet`(13,964 根日线)。
- **分析脚本**：`src/yin_yang_analysis.py`。

## 三、方法

1. 对每一个交易日,用**收盘价对前一交易日收盘价**计算涨跌（close-to-close）。
2. 定义：当日收益 > 0 为"涨天"；< 0 为"跌天"；== 0 为"平盘"。
3. 按自然年分组统计涨/跌/平盘天数,计算涨/跌比例与对 1:1 的偏离。
4. 全部使用实际获取的数据,不进行任何插值、估算或假设。

## 四、总体结论 —— 假设不成立

**纳斯达克综合指数 56 年间的累计数据**：

| 指标 | 数值 |
|---|---|
| 总交易日 | 13,964 |
| 上涨天数 | 7,787 (55.76%) |
| 下跌天数 | 6,139 (43.96%) |
| 平盘天数 | 38 |
| **涨/跌比例** | **1.268 : 1** |
| 平均每年偏离 1:1 的程度 | 13.95 个百分点 |
| 上涨天数多于下跌天数的年份 | 49/56 |
| 下跌天数多于上涨天数的年份 | 7/56 |

**关键判定**：累计涨/跌比例约为 **1.27 : 1**（不是 1:1）,平均每年偏离 1:1 达 **13.95 个百分点**。
**"上涨天数 = 下跌天数"的阴阳平衡假设,在纳斯达克综合指数 56 年历史数据上不成立。**

## 九、结论与推论

1. **假设不成立**：纳斯达克综合指数 56 年累计上涨天数(7,787)比下跌天数(6,139)多 27%,整体比例约 1.27:1。
2. **多数年份明显偏向"涨多跌少"**:49 个年份上涨天数占比超过 50%,仅 7 个年份下跌天数占比超过 50%。
3. **牛熊市偏离方向相反**：
   - 牛市（如 1979、1980、1989、2020）涨天比例可达 60% 以上,极端牛市（如 1979 年 66%）偏离 1:1 超过 30 个百分点。
   - 熊市（如 2000、2001、2008）跌天占比上升,但即使在大跌年份,跌天也很少超过 55%。
4. **阴阳平衡并非自然状态**：美股作为长期向上的资产类别,其内在结构是"涨多跌少"而非"涨跌平衡"。假设更接近"熊市短暂、牛市长"而非"涨跌各半"。
5. **可操作启示**：如果用"涨/跌天数接近 1:1"作为交易系统入场信号,会产生大量假信号（因为整体偏向涨天）。更稳健的入场信号应该是"回调日连续累积且偏离值远低于该年历史均值"。

## 十、数据局限

1. 仅用收盘对收盘定义涨跌,未考虑日内（开盘到收盘）的"实际上涨天数",会有微小差异。
2. 1971 年数据从 2 月开始,不是完整自然年。
3. 未做指数成分调整（survivorship）,但指数本身按编制规则调整,影响极小。
