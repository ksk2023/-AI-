# 研究报告索引

本目录存放所有量化研究输出,所有数据来自 Yahoo Finance 复权日线(可复现)。

## 命名规范

- 主题报告:`YYYY-MM-DD_研究主题.md`(本目录顶层)
- 个股报告:`个股分析/<TICKER>/YYYY-MM-DD_<TICKER>_历史回测分析.md`
- 同主题不覆盖,按日期形成时间线;后续更新改日期即可
